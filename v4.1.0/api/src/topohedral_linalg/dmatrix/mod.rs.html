1//! Dynamic matrix type with heap-allocated, column-major storage.
2//!
3//! Defines [`DMatrix<T>`], a general-purpose 2-D matrix whose dimensions are determined at
4//! runtime and whose elements are stored in a contiguous `Vec<T>` in column-major (Fortran) order.
5//! Sub-modules add element-wise arithmetic ([`addop`], [`subop`], [`mulop`], [`divop`], [`negop`]),
6//! BLAS-backed matrix multiplication ([`matmul`]), standard linear-algebra decompositions
7//! ([`lu`], [`qr`], [`eig`], [`symeig`], [`schur`], [`solve`]), and supporting utilities for
8//! construction, indexing, iteration, I/O, sub-matrix views, and reduction/transformation
9//! operations.
