1//! Thin, type-safe wrappers around BLAS Level 2/3 and LAPACK routines.
2//!
3//! Groups all raw BLAS and LAPACK bindings used by the crate into named sub-modules, each
4//! corresponding to a single subroutine family. The wrappers expose Rust traits rather than raw
5//! `unsafe` function pointers, so callers work with typed inputs and outputs and `unsafe` blocks
6//! are confined to the implementation files. Sub-modules: [`common`] (shared types), [`gemm`]
7//! (matrix–matrix multiply), [`gemv`] (matrix–vector multiply), [`getrf`] (LU), [`geqrf`] +
8//! [`orgqr`] (QR), [`geev`] (general eigenvalues), [`syev`] (symmetric eigenvalues), [`gees`]
9//! (Schur), [`gesv`] (linear solve).

34{
