1//! # Welcome to TopoHedral-Linalg!
2//!
3//! This crate provides a Rust library for small, dense linear algebra. It provides
4//! two matrix types:
5//!
6//! - a runtime-sized `DMatrix`
7//! - a compile-time-sized `SMatrix`
8//!
9//! Both use column-major memory layout and can hold both floating point and integral data.
10//! Both have the following features defined for them:
11//!
12//! - Accelerated matrix-matrix and matrix-mector multiplication via BLAS/LAPACK (floating point only)
13//! - Accelerated Matrix decompositions and linear system solution via BLAS/LAPACK (floating point only)
14//! - Lazily-evaluated, complex elementwide expressions.
15//! - Matrix subviews
16//! - Reductions and transformations
17//! - Elementwise functions which mirror those supported for primitive integral and floating point
18//!   types.
19//!
20//! For guides, tutorials, examples and developer documentation, see the
21//! [TopoHedral-Linalg documentation site](../../).
