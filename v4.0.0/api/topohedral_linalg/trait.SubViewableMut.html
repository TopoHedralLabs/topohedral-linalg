Expand description

Mutable subview constructor methods for any matrix type.

Extends SubViewable with IndexMut<(usize, usize)>. Only subview_mut must be provided; the remaining methods have defaults.

Default methods bind dimension reads to locals before the &mut self call to avoid simultaneous shared + exclusive borrow of self.
