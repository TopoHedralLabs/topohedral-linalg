Struct DMatrix

pub struct DMatrix<T>
where
    T: Copy,

Expand description

A dynamic-size $N \times M$ matrix type that stores its elements in a dynamic, contiguous array.

The DMatrix struct represents a 2D matrix with a dynamic size, where the dimensions are specified as generic parameters N and M. The elements of the matrix are stored in a contiguous array, which allows for efficient access and manipulation.

The matrix is stored in column-major order, which means a matrix is stored column by column in memory. So, for example, the matrix:

ⓘ
1 2 3
4 5 6
7 8 9

will be stored in memory as:

ⓘ
1 4 7 2 5 9 3 6 9

pub fn cholesky(&self) -> Result<Return<T>, Error>

Computes the Cholesky decomposition of the matrix.

Factors self into L such that self = L L^T. Only the lower triangle of self is read by LAPACK.

§Errors

Returns Error::PotrfError if LAPACK potrf fails, including when the matrix is not positive definite.

pub fn eig(&self) -> Result<Return<T>, Error>

Computes the eigendecomposition of a general square matrix.

Returns all eigenvalues and both left and right eigenvectors of self. Eigenvalues are represented as complex numbers even when the input matrix is real-valued.

§Errors

Returns Error::GeevError if the LAPACK geev routine fails.

pub fn lu(&self) -> Result<Return<T>, Error>

Computes the LU decomposition of the matrix.

Factors self into P, L, and U such that PA = LU, where P is a permutation matrix, L is lower-triangular with unit diagonal, and U is upper-triangular.

§Errors

Returns Error::GetrfError if the LAPACK getrf routine fails.

pub fn qr(&self) -> Result<Return<T>, Error>

Computes the QR decomposition of the matrix.

Factors self into Q and R such that A = QR, where Q is orthogonal and R is upper-triangular.

§Errors

Returns Error::GetrfError if geqrf fails, or [Error::OrgqrError] if orgqr fails.

pub fn schur(&self) -> Result<Return<T>, Error>

Computes the Schur decomposition of the matrix.

Factors self into Q and T such that A = Q T Q^H, where Q is orthogonal and T is quasi-upper-triangular.

§Errors

Returns Error::GeesError if the LAPACK gees routine fails.

pub fn solve(&self, b: &DMatrix<T>) -> Result<DMatrix<T>, Error>

Solves the linear system A X = B for X.

Uses LAPACK gesv, which performs LU factorisation with partial pivoting on self in order to compute the solution matrix X. self must be square and non-singular.

§Errors

Returns Error::GesvError if the LAPACK gesv routine fails.

pub fn sort(&mut self, dim: Dimension)

Sorts the elements of the matrix in-place along the specified dimension.

When dim is Dimension::Rows, each row is sorted independently. When dim is Dimension::Cols, each column is sorted independently. When dim is Dimension::All, all elements are sorted as a single sequence.

Source

pub fn sorted(&self, dim: Dimension) -> Self

Returns a new matrix with elements sorted along the specified dimension, leaving self unchanged.

See sort for the semantics of dim.

Source

pub fn into_sorted(self, dim: Dimension) -> Self

Consumes self, sorts its elements along the specified dimension, and returns the result.

Prefer this over sorted when the original matrix is no longer needed, as it avoids an extra allocation.
